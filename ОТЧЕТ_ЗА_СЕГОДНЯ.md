## Возможные причины:

1. **Локальные файлы не синхронизированы с сервером**
   - Выполните `git pull origin main` для получения свежих данных с сервера

2. **Бот не совершал сделок сегодня**
   - Проверьте логи на сервере: `tail -100 logs/trading_bot.log`

3. **Audit-логи на сервере более свежие**
   - Запустите скрипт синхронизации логов с сервера

## Структура отчета (когда появятся данные):

Отчет будет содержать:
- **Время операции** (по МСК)
- **Тип операции** (ПОКУПКА/ПРОДАЖА)
- **Символ**
- **Количество** (в штуках)
- **Цена**
- **Сумма операции**
- **Баланс кошелька** на момент операции
- **Прибыль/убыток** по каждой позиции
- **Общая статистика** за день

## Созданный скрипт:

Файл `report_today_trades.py` готов к использованию и автоматически:
- Определяет сегодняшний день по МСК
- Фильтрует сделки за сегодня
- Вычисляет прибыль/убыток
- Форматирует отчет

Запустите его после синхронизации данных с сервера.
