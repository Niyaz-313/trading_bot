# БЫСТРАЯ УСТАНОВКА АВТОСИНХРОНИЗАЦИИ ЛОГОВ

## Шаг 1: Отправить файлы на сервер

На локальном ПК выполните:
```batch
cd "C:\Users\karim\OneDrive\Рабочий стол\Bot"
git add server_sync_logs_to_git.sh setup_cron_sync.sh ИНСТРУКЦИЯ_АВТОСИНХРОНИЗАЦИЯ_ЛОГОВ.md
git commit -m "Добавлена автоматическая синхронизация логов с Git"
git push origin main
```

## Шаг 2: Установить на сервере

На сервере выполните:
```bash
cd /home/botuser/trading_bot/trading_bot

# Получить новые файлы
git pull origin main

# Установить права и запустить установку
chmod +x setup_cron_sync.sh
chmod +x server_sync_logs_to_git.sh
./setup_cron_sync.sh
```

## Шаг 3: Проверка

```bash
# Проверить cron задачу
crontab -l | grep sync_logs

# Запустить синхронизацию вручную для проверки
./server_sync_logs_to_git.sh

# Проверить логи
tail -f logs/git_sync.log
```
