========================================
МОНИТОРИНГ ЛОГОВ БОТА В РЕАЛЬНОМ ВРЕМЕНИ
========================================

ЧТО ВЫ ВИДИТЕ: Это просто просмотр файла через less/cat
ВЫХОД: Нажмите 'q' или Ctrl+C

========================================
ОСНОВНЫЕ КОМАНДЫ
========================================

# Просмотр логов в реальном времени (РЕКОМЕНДУЕТСЯ)
sudo journalctl -u trading-bot.service -f

# Просмотр последних 50 строк и затем в реальном времени
sudo journalctl -u trading-bot.service -n 50 -f

# Просмотр логов файла в реальном времени
tail -f /home/botuser/trading_bot/trading_bot/logs/trading_bot.log

# Просмотр логов ошибок
tail -f /home/botuser/trading_bot/trading_bot/logs/bot_error.log

========================================
ПРОВЕРКА СТАТУСА
========================================

# Статус сервиса
sudo systemctl status trading-bot.service

# Статус без пагинации
sudo systemctl status trading-bot.service --no-pager

========================================
ПОСЛЕДНИЕ ЛОГИ
========================================

# Последние 100 строк
sudo journalctl -u trading-bot.service -n 100

# Логи за последний час
sudo journalctl -u trading-bot.service --since "1 hour ago"

========================================
ПОИСК В ЛОГАХ
========================================

# Поиск ошибок
sudo journalctl -u trading-bot.service | grep -i error

# Поиск конкретного символа
sudo journalctl -u trading-bot.service | grep "SBER"

========================================
ВЫХОД ИЗ ПРОСМОТРА
========================================

- Нажмите 'q' (quit) - для выхода из less
- Нажмите Ctrl+C - для остановки tail -f или journalctl -f
