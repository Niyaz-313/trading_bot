========================================
КОМАНДЫ ДЛЯ ВЫПОЛНЕНИЯ НА СЕРВЕРЕ
========================================

ЗАМЕНИТЕ YOUR_IPv4_ADDRESS на ваш IPv4 адрес сервера!

========================================
1. ПОДКЛЮЧЕНИЕ К СЕРВЕРУ
========================================

На локальной машине (PowerShell):
ssh botuser@YOUR_IPv4_ADDRESS

Пример:
ssh botuser@192.168.1.100

========================================
2. КЛОНИРОВАНИЕ GIT РЕПОЗИТОРИЯ
========================================

На сервере:
cd ~
git clone https://github.com/Niyaz-313/trading_bot.git trading_bot
cd trading_bot

========================================
3. УСТАНОВКА ЗАВИСИМОСТЕЙ
========================================

На сервере:
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt

========================================
4. КОПИРОВАНИЕ .ENV ФАЙЛА
========================================

На локальной машине (PowerShell):
scp "C:\Users\karim\OneDrive\Рабочий стол\Bot\.env" botuser@YOUR_IPv4_ADDRESS:~/trading_bot/.env

Пример:
scp "C:\Users\karim\OneDrive\Рабочий стол\Bot\.env" botuser@192.168.1.100:~/trading_bot/.env

========================================
5. ОБНОВЛЕНИЕ КОДА НА СЕРВЕРЕ
========================================

На сервере:
cd ~/trading_bot
source venv/bin/activate
git pull origin main
pip install -r requirements.txt

========================================
6. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
========================================

На сервере:
cd ~/trading_bot
source venv/bin/activate
python3 -c "from broker_api import BrokerAPI; b = BrokerAPI(paper_trading=True); print('OK' if b.client else 'FAIL')"

========================================
ПРИМЕЧАНИЯ
========================================

- Все команды выполняются на сервере после подключения по SSH
- IPv4 адрес указывается БЕЗ квадратных скобок (в отличие от IPv6)
- Если репозиторий приватный, настройте авторизацию (см. ПОДКЛЮЧЕНИЕ_GIT_К_СЕРВЕРУ.md)
